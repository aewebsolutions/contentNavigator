Demo: Collapsable

Turning true collapsable option, navigator's nested titles will be easily hidden and shown. Also, if you turn autocollapse true, titles will be toggle visibility automatically when you scroll through page's contents.
If you like to use collapsable, navigator should be absolute or fixed positioned. Otherwise, toggling visibility would affect contents blocks' offset and neither links nor URL hash will be properly work.
By default, collapse button is an arrow. Click on it to toggle visibility. Also, you can modify it through css or even you can change the original markup (see btnCollapse option) 


$('#navigator1').contentNavigator({
    wrapper : '#demo1',
    collapsable : true,
    autocollapse : true,
    updateURL : true
});
