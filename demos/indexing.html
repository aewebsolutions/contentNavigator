Adding index numbers

    wrapper : '#demo1',
    indexOnContent : true

Index type

use numbers and letters to specify an index type notation. Start with a dot if you want to leave empty first level. 


$('#navigator2').contentNavigator({
    wrapper : '#demo2',
    indexOnContent : true,
    indexType : '.A.1.a',
    nestedIndex : false
});
